# Font del gioco

Il gioco (`../restaurant_game.html`) usa due famiglie:

- **Poppins** (corpo del testo) — ✅ già presenti qui, licenza SIL Open Font License (vedi `OFL.txt`), libere da ospitare anche su URL pubblico:
  - `Poppins-Regular.ttf`, `Poppins-Medium.ttf`, `Poppins-SemiBold.ttf`, `Poppins-Bold.ttf`

- **Oddval Text** (titoli, font brand Datapizza) — ⚠️ **da inserire qui** (non scaricabili pubblicamente). Nomi file attesi dal CSS:
  - `OddvalText-Regular.otf`
  - `OddvalText-Medium.otf`
  - `OddvalText-SemiBold.otf`
  - `OddvalText-Bold.otf`
  - `OddvalText-Black.otf`

Se i file Oddval non vengono inseriti, il gioco resta perfettamente funzionante: i titoli ripiegano su Poppins / font di sistema (nessun errore).
